# Alethia

Another manga reader for iOS built with SwiftUI (and some UIKit)

## Overview

Another generic manga reader that takes features, designs and functionality from other readers that I like.

### Technology Stack

- **Language**: Swift 6
- **UI**: SwiftUI (UIKit)
- **Database**: GRDB (SQLite) + Combine ValueObservations

### Requirements

Last ran on the following:

- iOS 18.0
- Xcode 16.1
- Swift 6

### Architecture

'Screens' use standard MVVM architecture.
